# DBX Git Behavior Exploration

This repository is used to explore Databricks Git behavior.

## Test files

- `notebooks/dbx_diff_test.ipynb`: notebook used for cell-based diff testing
- `sample.py`: Python file used for normal file diff testing

## Scenarios to test

- Modify a notebook code cell
- Modify a notebook markdown cell
- Add a new notebook cell
- Delete a notebook cell
- Reorder notebook cells
- Modify a normal Python file
